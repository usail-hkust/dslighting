# 房价预测挑战

## 任务描述

预测给定房屋的销售价格（美元）。这是一个典型的回归任务，需要基于房屋的特征（面积、房间数、位置等）来估算其市场价格。

## 数据说明

### 文件
- `train.csv` - 训练数据，包含房屋特征和实际价格
- `sample_submission.csv` - 提交格式示例

### 特征列

| 列名 | 类型 | 描述 | 范围 |
|------|------|------|------|
| `house_id` | int | 房屋唯一标识符 | 1 - N |
| `area` | int | 居住面积（平方英尺） | 800 - 3500 |
| `bedrooms` | int | 卧室数量 | 1 - 5 |
| `age` | int | 房屋年龄（年） | 0 - 50 |
| `location_score` | int | 位置评分 | 1 - 10 |
| `price` | float | 销售价格（美元） | 仅训练集 |

## 提交格式

提交文件必须是 CSV 格式，包含以下列：

```csv
house_id,predicted_price
1,250000.00
2,350000.00
...
```

- `house_id`: 房屋 ID（与测试集对应）
- `predicted_price`: 预测的价格（浮点数）

## 评分标准

**评分指标**: RMSE（Root Mean Squared Error）

```
RMSE = sqrt(mean((predicted_price - actual_price)^2))
```

- **越低越好**：RMSE 为 0 表示完美预测
- 提交的预测将与私有测试集的实际价格进行比较

## 任务目标

最小化预测价格与真实价格之间的 RMSE，构建最准确的房价预测模型。

## 难度级别

**中等** - 适合学习回归算法和特征工程
