# Contributing

Thanks for helping improve DSLIGHTING.

## Setup
- Use Python 3.10+
- Create a virtualenv and install requirements

## Code style
- Keep changes focused and minimal
- Prefer small, reviewable commits
- Add or update docs when behavior changes

## Testing
- Run relevant scripts for the workflow you changed
- Include repro steps in your PR/issue

## Submitting changes
- Describe what changed and why
- Include any new or updated configs
