# 核心功能

DSLIGHTING 提供了完整的数据科学工作流自动化能力。

## Agent 工作流

### 🤖 AIDE
迭代式代码生成和审查循环，通过持续的代码改进来优化模型性能。

### 🧠 Automania
带记忆和任务分解的规划+推理系统，能够处理复杂的多步骤任务。

### 🎯 DSAgent
结构化操作符流程的规划/执行循环，提供清晰的任务分解和执行策略。

### ⚡ Data Interpreter
快速代码执行和调试循环，专注于代码的快速迭代和错误修复。

### 🏆 AutoKaggle
SOP风格的Kaggle工作流，专为数据竞赛优化。

### 🔄 AFlow
工作流的元优化，自动选择和组合最优的工作流策略。

### 🔬 DeepAnalyze
专注分析型执行工作流，深入分析数据和模型性能。

## 核心特性

### 📊 交互式 Web 界面
- 基于 Next.js + FastAPI 构建
- 实时任务进度监控
- 可视化结果展示
- 支持自定义任务配置

### 📝 完整日志追踪
- 记录完整的执行轨迹
- 保存所有中间产物
- 支持结果复现和审查
- 自动生成性能报告

### 🧩 可扩展架构
- 灵活的任务注册机制
- 支持自定义工作流
- 可插拔的Agent系统
- 丰富的Benchmark支持

### 🎯 自动化评估
- 多种评估指标
- 自动模型选择
- 性能基准测试
- A/B测试支持

## 数据流程

\`\`\`
数据准备 → 特征工程 → 模型训练 → 模型评估 → 结果生成
    ↓         ↓          ↓          ↓          ↓
Agent工作流贯穿全流程，提供智能决策和自动化执行
\`\`\`

## 支持的 Benchmark

- **MLE-Bench**: OpenAI机器学习评估基准
- **Kaggle**: 支持多种Kaggle竞赛
- **自定义任务**: 支持用户自定义数据集和任务
